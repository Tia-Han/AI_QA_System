# 小型人工智能基础问答系统（V1.0）

## 1. 项目概述

### 项目名称

小型人工智能基础问答系统

### 项目背景

本项目为《数据结构与算法》课程实验，基于 Python 开发一个简单的人工智能基础知识问答系统，通过预设知识库、关键词匹配和 Web 页面交互，实现基础问答功能，并掌握 Python 数据结构及工程化开发流程。

---

## 2. 产品目标

实现一个支持人工智能基础知识查询的小型问答系统。

系统应具备以下能力：

* 构建20条人工智能基础知识库
* 支持用户输入问题
* 根据关键词匹配返回最佳答案
* 保存用户历史提问
* 提供 Streamlit 可视化网页
* 使用 Git 管理项目

---

## 3. 功能需求

### 功能一：知识库管理

使用 Python Dictionary 构建知识库。

每条数据包含：

* question：问题
* answer：答案
* keywords：关键词列表

知识库共20条数据。

---

### 功能二：问答匹配

流程：

用户输入问题

↓

提取关键词

↓

遍历知识库

↓

计算关键词交集数量

↓

返回交集最多的问题答案

若没有匹配结果，则提示：

> 抱歉，没有找到相关答案，请尝试其他问题。

---

### 功能三：用户交互

采用 Streamlit 开发网页。

页面包含：

* 系统标题
* 问题输入框
* 查询按钮
* 回答区域
* 历史记录区域

支持连续提问。

---

### 功能四：历史记录

使用 List 保存用户所有提问。

页面实时展示历史提问记录。

---

## 4. 数据结构设计

| 数据结构       | 用途         |
| ---------- | ---------- |
| Dictionary | 存储知识库      |
| List       | 保存历史记录     |
| Set        | 关键词去重及交集计算 |
